# Todo MVC Application

A simple, clean Todo application built with Flask and SQLite. This application allows users to create, manage, and track their tasks with a straightforward interface.

## Technologies Used

- Python 3.x
- Flask (Web Framework)
- SQLite (Database)
- HTML/CSS
- Bootstrap (Styling)
